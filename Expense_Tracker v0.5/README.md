# About Expense Tracker v0.5

This version of *Personal Expense and Budget Tracker* allows you to:
- Add the category, amount, note of your expenses and automatically assigns the date you entered it with a unique index for each category.
- Edit value under any column of any expense you want.
- Delete any row/expense.
- Delete all the values of any column (i.e. category, amount, note) you want,
  which you can later assign new values individually via *Edit Category* if you don't want to keep them empty.
- Compare/see your expenses in 7 different types of graph that are available.
- See the total amount you spent or in any given time frame.
